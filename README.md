QUE ES DEVOPS?
Es la forma en la que trabajan un conjunto de desarrolladores en un proyecto, donde están involucrados quienes desarrollan el codigo como los de infraestructura (servidores, redes, despliegues) colaboran y se coordinan para que el software se desarrolle, pruebe y publique más rápido y sin tantos errores.
Es como si los de desarrollo y los de sistemas ya no se tiraran la pelota de quién dañó qué, sino que trabajaran juntos como un solo equipo.

DEFINICION:
DevOps es una cultura y conjunto de prácticas que automatiza e integra los procesos entre el desarrollo de software y las operaciones de TI, permitiendo que las aplicaciones se entreguen más rápido, de forma continua y con mayor calidad.
